# Glossary

Workplace shorthand, acronyms, and internal language for the WaaS project.

## Acronyms

| Term | Meaning | Context |
|------|---------|---------|
| WaaS | Workforce-as-a-Service | The platform — cognitive agent orchestration |
| EA | Executive Assistant | Agent "Alex", human-facing primary contact |
| CFA | Chief Financial Agent | Agent "Morgan", financial modeling + forecasting |
| COA | Chief Operating Agent | Agent "Jordan", operations + agent management |
| CMA | Chief Marketing Agent | Agent "Taylor", content + brand + campaigns |
| IR | Investor Relations | Agent "Riley", under CFA, investor pipeline |
| RLS | Row Level Security | Supabase access control, org-scoped |
| SSE | Server-Sent Events | Streaming protocol for chat responses |
| MCP | Model Context Protocol | Tool protocol for Claude agents |
| RPC | Remote Procedure Call | Supabase atomic functions (e.g., create_organization) |
| TTS | Text-to-Speech | ElevenLabs voice synthesis |
| STT | Speech-to-Text | ElevenLabs voice recognition |
| VAD | Voice Activity Detection | Turn-taking in voice conversations |
| JWT | JSON Web Token | Auth tokens from Supabase |
| MRR | Monthly Recurring Revenue | Derived metric computed by CFA tools |
| CF | Cloudflare | AI Gateway, Workers, Queues |
| AIG | AI Gateway | Cloudflare's API proxy for caching/analytics |
| DO | DigitalOcean | Hosting platform for agent servers |

## Internal Terms

| Term | Meaning |
|------|---------|
| agent network | The full set of cognitive + conversational AI agents |
| agentic loop | Claude API tool-use loop (max 15 turns for EA) |
| tool bridge | Pattern where tools are native Anthropic API defs + handlers |
| enrichment pipeline | Parallel memory + plugin loading before each agent query |
| namespace isolation | ScopedRedisClient + ScopedMemoryClient per department |
| fail-closed | Unregistered agents denied by default (not allowed) |
| dual-mode | Cognitive (Claude + tools) + Conversational (ElevenLabs voice) |
| cognitive only | Agent has reasoning + tools but no voice capability |
| conversational only | Voice-only agent (e.g., sales reps), no deep reasoning |
| org-scoped | Data access restricted to a single organization via RLS |
| board session | Multi-agent deliberation with quorum voting |
| message bus | Inter-agent messaging via Redis LISTs + Telegram |
| hot cache | CLAUDE.md Tier 1 memory (top 30 items) |
| deep memory | memory/ directory Tier 2 (full knowledge base) |
| data room | Public investor-facing document portal |
| provider keys mode | CF AI Gateway injects API keys at edge |
| sales swarm | 10 conversational sales reps making batch calls |
| custom_llm | ElevenLabs endpoint routing voice to our cognitive /chat |
